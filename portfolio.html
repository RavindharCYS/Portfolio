This project explores an innovative method for generating and storing renewable energy from refrigerators using the Peltier effect. By leveraging semiconducting materials, the system converts waste heat from refrigeration into electricity, producing a stable 12V output for battery storage or direct use. The generated DC power is transformed into AC and boosted to 240V, sufficient for commercial applications. The system is designed to reduce reliance on traditional power grids, lower electricity costs, and contribute to environmental sustainability. The inclusion of solar panels eliminates the need for external power sources, creating a self-sustaining, pollution-free energy solution.

SentinelSight is a Python-based Digital Forensic Analysis Tool designed to enhance cybersecurity through advanced SIEM (System Information and Event Management) capabilities. By centralizing and analyzing log data in real-time, it enables efficient threat detection, incident response, and compliance monitoring. Incorporating machine learning and behavioral analytics, SentinelSight improves detection accuracy and speeds up incident investigations. Its comprehensive reporting tools help organizations meet regulatory requirements and mitigate risks, providing a robust defense against evolving cyber threats.

The Password Generator and Manager web application provides a secure, user-friendly solution for password management, built exclusively with HTML, CSS, and JavaScript. It offers three key functionalities: password generation, secure storage, and password retrieval. Using client-side JavaScript, the application generates strong, customizable passwords and encrypts them locally with a master password, ensuring user privacy. The intuitive interface enables easy navigation and access to all features, while responsive design ensures compatibility across devices. By eliminating server-side storage, the application enhances security, giving users complete control over their password data.
